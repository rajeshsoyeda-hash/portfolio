# 🚀 Rajesh Soyeda — Personal Portfolio

A modern, full-stack personal portfolio website built to showcase my projects, skills, and experience.

🔗 **Live Site:** <https://rajeshsoyeda-hash.github.io/portfolio/>
---

## ✨ Features

- 🎨 Dark theme with smooth animations
- 📱 Fully responsive design
- 🗄️ LocalStorage-based project database (CRUD)
- 📬 Contact form with message storage
- ⚡ Zero dependencies — pure HTML, CSS, JS

---

## 🛠️ Tech Stack

| Layer | Technology |

| Frontend | HTML5, CSS3, Vanilla JavaScript |
| Animations | CSS Keyframes, Intersection Observer API |
| Database | localStorage (JSON-based) |
| Deployment | GitHub Pages |

---

## 📂 Project Structure

- index.html — main portfolio layout and content
- style.css — visual styling, layout, and animations
- script.js — interactive effects and animations
- README.md — project overview and setup instructions

---

## 🚀 Run Locally

```bash
# Clone the repo
git clone https://github.com/rajeshsoyeda-hashsename/portfolio.git

# Open the project folder
cd portfolio-main

# Open the portfolio in your browser
start index.html
```
